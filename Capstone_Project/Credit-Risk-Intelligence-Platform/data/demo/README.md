# Demo Documents

Place sample financial PDFs here to test the platform without real documents.

## What to use for testing

Any publicly available annual report PDF works well:

- **HDFC Bank Annual Report** — search "HDFC Bank annual report 2023 PDF"
- **SBI Annual Report** — search "SBI annual report 2023 PDF"  
- **Infosys Annual Report** — search "Infosys annual report 2023 PDF"
- **Any company balance sheet or P&L statement**

## For individual/borrower testing

To test the Retirement Planner feature, use documents that contain:
- Monthly income figures
- Expense data
- Loan/liability information

A simple Word-to-PDF bank statement works fine for testing.

## File naming tips

Name files descriptively — the platform auto-detects document type from the name:
- `HDFC_Annual_Report_2023.pdf` → detected as Annual Report
- `SBI_Balance_Sheet_2023.pdf` → detected as Balance Sheet
- `ITR_FY2023.pdf` → detected as Income Tax Return

## Notes

- Files in this folder are NOT auto-loaded — you must upload via the UI
- For testing: a 5-10 page PDF processes in about 30-60 seconds
- Larger PDFs (100+ pages) may take 2-3 minutes
