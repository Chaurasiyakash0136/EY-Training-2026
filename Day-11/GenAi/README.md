# Day-11 GenAI Assignment

## Objective

This project demonstrates:

1. Zero-Shot Summarization
2. Few-Shot Summarization
3. ROUGE-L Evaluation
4. 5-Class Ticket Classification
5. Chain-of-Thought Style Reasoning

## Technologies

* Python
* LangChain
* Groq
* PromptTemplate
* ROUGE-L

## Tasks

### Task 1

Summarize 3 earnings call transcripts using:

* Zero-Shot Prompting
* Few-Shot Prompting

### Task 2

Evaluate generated summaries using:

* ROUGE-L

### Task 3

Classify tickets into:

* Billing
* Tech
* Refund
* General
* Escalate

with reasoning.

## Execution

```bash
pip install -r requirements.txt

python main.py
```

## Outputs

Generated automatically:

* summaries.json
* rouge_scores.csv
* ticket_predictions.json
